# News Finder App

This is a React application designed to fetch and display relevant news articles based on user queries. 

## Description

The app interfaces with an API to retrieve news articles pertinent to what the user wants to find. With a user-friendly interface and fast retrieval mechanism, this app ensures that users get the most relevant news pieces quickly and efficiently.

## Live Demo

Experience the app live at [https://21news.netlify.app/](https://21news.netlify.app/)

## Features

- **Search Functionality**: Allows users to search for news by entering keywords.
- **Responsive Design**: Optimized for both desktop and mobile views.
- **Fast Retrieval**: Efficiently fetches news articles from the backend.
